# Шаблон лабораторных работ и курсовых проектов кафедры ИСПИ

Этот репозиторий содержит Typst-шаблон для оформления лабораторных работ и курсовых проектов студентов кафедры информационных систем и программной инженерии (ИСПИ) ВлГУ.

## Установка

### Через локальный пакет (для разработчиков)

Для удобной разработки шаблона можно установить его как локальный пакет Typst. Это позволяет использовать короткий импорт:

```typst
#import "@local/vlsu-report:0.1.0": template
```

**Установка:**

1. Клонируйте репозиторий или скачайте файлы
2. Запустите скрипт установки:

```powershell
.\update-package.ps1
```

Скрипт скопирует необходимые файлы в `%LOCALAPPDATA%\typst\packages\local\vlsu-report\<версия>\`.

**Обновление локального пакета:**

При внесении изменений в шаблон запустите скрипт снова — он обновит локальный пакет:

```powershell
.\update-package.ps1
```

Версия берётся автоматически из `typst.toml`. Если меняется версия — обновите её в `typst.toml`, затем запустите скрипт.

> **Примечание:** Скрипт предназначен для разработчиков шаблона. Студенты могут просто копировать файлы шаблона в свой проект.


## Быстрый старт

### Требования

- Typst 0.8.0 или выше

1. Скопируйте файлы шаблона в вашу директорию проекта
2. Создайте файл `main.typ` с вашей работой
3. Импортируйте шаблон:

```typst
#import "lib.typ": template

#show: template.with(
  title: [ Лабораторная работа №1 ],
  discipline: [
    по дисциплине \
    "Название дисциплины" \
    на тему
  ],
  theme: [ Тема работы ],
  students: (
    (
      name: "И. И. Иванов",
      group: "ИСТ-123",
      gender: "male"
    )
  ),
  teacher: (
    name: "П. П. Петрова",
    position: "ст. преп. каф. ИСПИ",
    gender: "female"
  ),
)

= Цель работы
...
```

4. Компилируйте: `typst compile main.typ`

## Структура документа

Каждая работа должна содержать следующие разделы:

### Лабораторная работа

```typst
= Цель работы
Описание цели (2-3 предложения)

= Выполнение работы

== Задание
Описание задания

== Другие разделы
Описание выполнения

= Вывод
Выводы по работе
```

### Курсовая работа

```typst
= Введение
Введение в предметную область

= Описание предметной области
Анализ предметной области

= Архитектура системы
Описание архитектуры

= Реализованные компоненты
Описание компонентов

= Заключение
Выводы и рекомендации

= Список использованных источников
Список литературы
```

## Оформление элементов

### Списки

Нумерованные списки:
```typst
+ Первый пункт
+ Второй пункт
+ Третий пункт
```

Маркированные списки:
```typst
- Первый пункт
- Второй пункт
- Третий пункт
```

### Изображения

```typst
#figure(
  image("path/to/image.png", width: 80%),
  caption: [Описание изображения]
)
```

### Таблицы

```typst

    [Ячейка 3], [Ячейка 4],

  caption: [Описание таблицы]
)
```

### Листинги кода

```typst
#figure(
```python

```,
  caption: [Описание кода]

## Рекомендации по оформлению

### Общие правила

- Используйте шрифт Times New Roman, размер 14pt
- Поля: левое 30mm, правое 15mm, верхнее и нижнее 20mm
- Межстрочный интервал: 1.5
- Первая строка абзаца: отступ 1.25cm
- Текст выравнивается по ширине

### Заголовки

- Заголовки 1-го уровня: центрированы, прописные буквы, жирный
- Заголовки 2-го уровня: центрированы, жирный
- Заголовки 3-го уровня: слева, жирный

### Нумерация страниц

- Первая страница (титульник): без номера
- Остальные страницы: номер в центре внизу

### Изображения и таблицы

- Все изображения и таблицы должны быть пронумерованы
- Подписи располагаются под изображением/таблицей
- Ссылки на изображения и таблицы в тексте обязательны

## Шрифты

Проект включает встроенные шрифты, поэтому дополнительная установка не требуется:

- **Times New Roman** — основной шрифт для текста документа
- **Courier New** — моноширинный шрифт для листингов кода
- **ГОСТ тип А и В** — шрифты для оформления рамок согласно ГОСТ

Все шрифты находятся в папке `fonts/` и автоматически используются при компиляции.

## Решение проблем

### Ошибки при компиляции

- Проверьте синтаксис Typst
- Убедитесь, что все импортируемые файлы находятся в правильных местах
- Проверьте кодировку файлов (должна быть UTF-8)

### Проблемы с кириллицей

Убедитесь, что файл сохранён в кодировке UTF-8. Язык уже установлен в шаблоне:

```typst
#set text(lang: "ru", font: "Times New Roman", size: 14pt)
```

### Warning про неизвестный шрифт

При компиляции может появиться warning про шрифт "ГОСТ тип А". Это нормально — система автоматически использует Times New Roman как fallback для основного текста.

## Дополнительные возможности

### Рамки для курсовых работ

Для курсовых работ доступны специальные рамки (ГОСТ). Все компоненты можно импортировать из `lib.typ`:

```typst
#import "lib.typ": template, frame, tables

#let frame-fields = (
  _title: [ Код проекта ],
  _description: [ Описание ],
  _group: [ Группа ],
  _developed: [ Разработчик ],
  _checked: [ Проверяющий ],
  _technical_controlled: [],
  _standard_controlled: [],
  _approved: [],
)

#show: template.with(...)
#show: frame.with(frame-fields, table: tables.form-2)
```

### Приложения

Для оформления приложений используйте функции `appendixes` и `appendix-heading`:

```typst
#import "lib.typ": appendixes, appendix-heading

#appendixes[
  #appendix-heading("обязательное")[Исходный код]

  == Модуль main
  #figure(
    ```python
    def main():
        print("Hello, World!")
    ```,
    caption: [Главный модуль]
  )

  #appendix-heading("справочное")[Техническая документация]

  == API справка
  Описание API методов.
]
```

### Нумерация заголовков и страниц

По умолчанию нумерация отключена. Включите её через параметры шаблона:

```typst
#show: template.with(
  title: [ Лабораторная работа №1 ],
  discipline: [ по дисциплине "Название" ],
  theme: [ Тема ],
  students: ((name: "И. И. Иванов", group: "ИСТ-123", gender: "male")),
  teacher: (name: "П. П. Петрова", position: "ст. преп.", gender: "female"),
  numbering-pages: true,    // Нумерация страниц
  numbering-headings: true, // Нумерация заголовков (1, 1.1, 1.1.1)
)
```

### Оглавление

Автоматическое оглавление:

```typst
#outline(
  title: [Содержание],
  depth: 2
)
```

### Счётчики

Автоматический подсчёт страниц, рисунков и таблиц:

```typst
Документ содержит #context(counter(page).final().first()) страниц,
#context(counter(image).final().first()) рисунков,
#context(counter(table).final().first()) таблиц.
```
